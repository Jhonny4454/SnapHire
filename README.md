# 📸 NA Photography Studio Booking Portal

A full-stack photography booking web application built using Flask and MySQL.

## 🚀 Features
- User Signup & Login
- Browse Photography Packages
- Add to Cart
- Checkout System
- Order History
- Profile Management

## 🛠️ Tech Stack
- Backend: Flask (Python)
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

## ⚙️ How to Run
1. Clone repo
2. Install dependencies:
   pip install flask mysql-connector-python
3. Run:
   python app.py
